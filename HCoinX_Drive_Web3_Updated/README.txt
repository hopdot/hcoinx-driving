This version connects the game directly to the HCoinX (HCX) token on Ethereum.

## Features
- Play driving game
- On crash, player enters wallet
- Game sends 5 HCX reward on-chain

## Setup

1. Install Python 3.10+
2. Create a .env file with:

ALCHEMY_URL=https://eth-mainnet.g.alchemy.com/v2/YOUR_KEY
GAME_WALLET_KEY=PRIVATE_KEY_WITH_HCX_BALANCE

3. Run build.bat to create EXE
4. Find EXE in /dist

## Controls
Left / Right arrows to move
